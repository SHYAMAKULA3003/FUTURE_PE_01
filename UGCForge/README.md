# 🚀 UGCForge — AI Ad Content Studio 

> Create ad content that feels real, connects emotionally, and converts better — powered by AI.

---

## 📌 Project Overview 

**UGCForge** is an AI-powered content generation platform designed to help creators, marketers, and businesses generate high-converting UGC-style ad content.

It focuses on producing **authentic, engaging, and platform-ready content** such as ad scripts, hooks, captions, and CTAs — optimized for modern social platforms.

This project demonstrates a strong understanding of real-world product building, combining UI/UX, backend logic, and AI integration.

---

## ✨ Core Features

* 🎯 **UGC Ad Content Generation**
  Generate scripts, hooks, captions, and CTAs instantly

* ⚡ **Fast Content Creation (< 30s)**
  Optimized for speed and responsiveness

* 📱 **Multi-Platform Ready**
  Content tailored for Instagram, TikTok, YouTube Shorts

* 🧠 **AI-Powered Engine**
  Smart content generation based on user inputs

* 📚 **Content Library**
  Store and manage generated content

* 🎨 **Modern UI/UX**
  Clean, engaging, and conversion-focused interface

## 🛠️ Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Database:** Prisma ORM
* **Backend:** Node.js services
* **Build Tools:** PostCSS
* **Package Manager:** npm / bun
